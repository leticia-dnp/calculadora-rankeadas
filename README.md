Este é meu 2° projeto do Bootcamp Santander e consiste em uma Calculadora de Partidas Rankeadas, desenvolvida em JavaScript. 
Ele recebe como entrada o número de vitórias e derrotas de um jogador, calcula o saldo de partidas (vitórias menos derrotas) e determina o nível do jogador com base nas vitórias, 
seguindo uma escala que vai de Ferro até Imortal. O intuito do projeto é a pratica da lógica de programação, utilizando conceitos de variáveis, operadores, estruturas de decisão, laços de repetição e funções,
além de consolidar habilidades em manipulação de dados e estruturação de código.
